# 数学核心：从自然数到不动点理论

> **兔狲教授的数学观**：数学不是一堆公式和定理的集合，而是一种思考世界的方式。它教会我们如何从简单的基本概念出发，通过严格的逻辑推理，构建出复杂而美妙的理论体系。这就是数学的力量——用确定性理解不确定性。

---

## 课程简介

**兔狲教授**在黑板上写下"数学 = 逻辑 + 想象力"，然后转身对你说：

> "很多人害怕数学，觉得它抽象、难懂。但数学的本质很简单：它只是用最精确的语言描述世界。自然数是我们对'数量'的直觉，集合论是我们对'分类'的抽象，函数是我们对'关系'的刻画，而不动点理论则揭示了'自我指涉'的奥秘。
>
> 这个课程就是要带你从最基础的直觉出发，一步步攀登数学的高峰。不用担心跟不上——我会用最生活化的比喻，让你看到数学背后的思想。"

**小小猪**举手提问："教授，为什么要从自然数开始？不能直接学微积分吗？"

**兔狲教授**微笑："好问题！就像建房子需要地基一样，理解高等数学需要坚实的基础。自然数公理是数学的'宪法'，集合论是数学的'通用语言'。没有这些基础，微积分就像空中楼阁。"

## 课程结构

### 第一模块：数学的基础语言
> 建立数学思维的基本框架

1. **[自然数与公理系统](./sections/natural-numbers.md)** - 数学的起点
   - 皮亚诺公理：自然数的定义
   - 数学归纳法：无穷的阶梯
   - 递归定义：自我构建的数学

2. **[集合论基础](./sections/set-theory.md)** - 数学的统一语言
   - 集合的基本操作
   - 关系与函数
   - 无穷集合的奥秘

3. **[逻辑与证明方法](./sections/logic.md)** - 数学推理的规则
   - 命题逻辑与谓词逻辑
   - 证明技巧：直接证明、反证法、数学归纳法
   - 公理化方法

### 第二模块：数学的核心结构
> 连接离散与连续的世界

4. **[函数与关系](./sections/functions.md)** - 数学结构的桥梁
   - 函数的类型：单射、满射、双射
   - 关系的性质：自反性、对称性、传递性
   - 等价关系与划分

5. **[数列与极限](./sections/sequences.md)** - 从离散到连续
   - 数列的收敛性
   - 极限的概念与计算
   - 柯西序列与完备性

### 第三模块：数学的深层理论
> 探索数学的基础与前沿

6. **[ZFC集合论](./sections/zfc.md)** - 数学的基础公理系统
   - 策梅洛-弗兰克尔公理
   - 选择公理及其等价形式
   - 集合论的哲学意义

7. **[不动点理论](./sections/fixed-points.md)** - 自我指涉的数学
   - 巴拿赫不动点定理
   - 布劳威尔不动点定理
   - 在计算机科学和经济学中的应用

## 学习建议

**小海豹**温和地建议："学习数学就像学习一门新语言，需要耐心和实践。我建议："

1. **循序渐进**：不要跳着学，每个概念都是下一个的基础
2. **动手练习**：数学是'做'出来的，不是'看'出来的
3. **联系实际**：思考每个概念在现实世界中的对应
4. **讨论交流**：和同学讨论，教别人是最好的学习方式

✅ **生活化比喻**：用果汁机比喻函数，用图书馆比喻集合

✅ **跨学科连接**：展示数学在计算机科学、经济学、物理学中的应用  
✅ **思维训练**：重点培养逻辑推理和问题解决能力
